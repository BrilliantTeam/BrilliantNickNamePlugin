# BrilliantNickNamePlugin / 簡單的暱稱插件

` /nickname help ` - 看怎麼用
` /nickname set ` - 設定暱稱，字符上限 20，不含色碼
` /nickname remove ` - 移除暱稱

使用以上指令的權限：` server.nickname `
更改他人暱稱的權限：` server.nickname.others `

---

` %nickname_get% ` - & 色碼
` %nickname_plain% ` - 無色碼
` %nickname_vanilla% ` - § 色碼

需要` PlaceholderAPI `，支援` Folia `環境。
